# GitHub User Finder

A simple web app to quickly search for any GitHub user and see their profile details.  

**Features:**
- Search by GitHub username
- Display avatar, name, login, bio, repos, followers & following
- Dark/Light mode toggle 🌙/☀️
- Shows last 5 recent searches for easy access
- Loading and error handling

**How to use:**
1. Type a GitHub username in the input box
2. Press **Enter** or click **Search**
3. View the profile card instantly
4. Switch themes using the 🌙 / ☀️ button
5. Click on recent searches to quickly search again
